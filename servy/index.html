Control LEDDY

Input commands in the following text box:


Usage: COMMAND [command opts ...] Commands: Text <text> ScrollingText <text> ClearDisplay FillDisplay Option text_spaceWidth <width> Option text_trailingWhitespace <spaces> Option updateDelayMs <milliseconds> Other client: HTTP Post to current URL with the command as plaintext body. End it with a newline.
Example: $ echo "ScrollingText Zeus WPI" | http POST 10.1.0.181
